# Retry mechanism

[![Build Status][1]][2]

Very simple retry mechanism with small dependency footprint.

## Maven repository

```xml
<dependency>
  <groupId>com.enigmabridge</groupId>
  <artifactId>retry</artifactId>
  <version>0.1.0</version>
</dependency>
```

## Gradle

```gradle
compile 'com.enigmabridge:retry:0.1.0'
```
